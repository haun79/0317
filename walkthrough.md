# 2D 足球遊戲實作成果

我已完成 2D 足球遊戲的開發。這是一個完全基於 HTML5 Canvas 與原生 JavaScript 的網頁遊戲，無需安裝任何外部套件。

## 實作特點

- **角色與視覺**:
    - [x] 球員身穿球衣球褲，具備頭部、身體、雙手與雙腳。
    - [x] 足球具有黑白五邊形圖案，渲染細緻。
    - [x] 足球場具備草地條紋、中場線、罰球區與立體感球門。
- **物理與碰撞**:
    - [x] 實作彈性碰撞偵測（球員 vs 球員，球員 vs 足球）。
    - [x] 加入摩擦力與速度限制，使移動具備慣性與平滑感。
    - [x] 足球邊界彈跳效果。
- **AI 行為 (終極智慧版)**:
    - [x] **50% 速度封印**: AI 極速被嚴格限制在玩家的 50%，確保對戰的公平性與成就感。
    - [x] **弧線向量移動**: 使用動態目標偏移與正弦波疊加，讓 AI 追球路徑呈現自然的「弧線」而非「直線」。
    - [x] **進階戰術思維**: AI 會根據 `frameCount` 動態重新計算攔截點，模擬真實人類的判斷過程。
    - [x] **區域減速場**: 接近球感應圈 (140px) 時會動態降低加速度，實作極其平滑的控球精準度。
- **場地視覺 (專業足球場)**:
    - [x] **森林綠草皮**: 回歸深淺相間的綠色條紋設計，符合國際足球場視覺標準。
    - [x] **高清 HUD**: 重新設計了具備毛玻璃效果 (Backdrop Blur) 的計分與計時介面。
    - [x] **結算動畫**: 加入了更平滑的勝負顯示頁面與互動按鈕。
- **遊戲規則**:
    - [x] 左右球門計分系統，分數更新顯眼。
    - [x] 進球後自動重置角色與球到初始位置。
